# Strategic Tags Strategy

## Approved Tag Set (7 tags maximum)

1. **announcement** - General announcements and official communications
2. **testing** - Testing phases, rounds, and testing-related content
3. **guide** - Tutorials, how-to guides, and step-by-step instructions
4. **project-42** - General Project 42 content
5. **infinite-improbability-drive** - Content specific to the Infinite Improbability Drive blockchain
6. **drive** - Content about Drive client tools
7. **genesis** - Content about genesis, chain launch, and initial setup

## Tag Usage Guidelines

- Use 2-4 tags per post maximum
- Always use English tags for consistency
- Combine tags strategically:
  - `testing` + `genesis` = Testing round about chain launch
  - `guide` + `drive` = Guide about using Drive
  - `announcement` + `project-42` = General project announcement
  - `testing` + `infinite-improbability-drive` = Testing specific to the blockchain

## Removed Tags

The following tags have been consolidated or removed:
- `round-1`, `chain-launch`, `validators` → Use `testing` + `genesis`
- `keys`, `security`, `preparation` → Use `testing` + `drive`
- `strategy` → Use `announcement` or `testing`
- `gentx` → Use `guide` + `genesis`
- `introduction` → Use `announcement`
